<h1 align="center">myDoro</h1>
<p align="center">
  献给自己的生日礼物，爱自己也爱生活
</p>

myDoro 是一份写给自己的生日礼物。它把“爱自己、爱生活”放在最前面：开发时陪在桌面上，替我看着 AI coding agent 的动静，也在忙碌间隙冒出来搞怪一下。

当 agent 开发过程中需要权限确认时，myDoro 会在桌面旁弹出提醒，帮我及时处理审批；平时它也会根据助手状态做出反应：提问时思考，跑工具时打字，子代理工作时杂耍，任务完成时庆祝，离开时安静睡觉。

支持 **Claude Code**、**Codex CLI** 和 **Copilot CLI**，三者可同时运行。

## 快速开始

```bash
git clone https://github.com/lezibo/myDoro.git
cd myDoro/myDoro
npm install
npm start
```
